Google Search Console — steps (you must complete in your browser)
================================================================

Prerequisite: https://sweitzerautomations.com loads with HTTPS, and
https://sweitzerautomations.com/sitemap.xml returns XML (not 404).
If the domain does not resolve, finish Netlify custom domain + DNS first.

IMPORTANT — property URL
------------------------
Add the SITE, not the sitemap file:
  CORRECT:   https://sweitzerautomations.com
  WRONG:     https://sweitzerautomations.com/sitemap.xml  (submit sitemap later, step 4)

If the property URL is wrong, verification URLs and the HTML file check will fail.

1) Go to https://search.google.com/search-console and sign in.

2) Add property → URL prefix → https://sweitzerautomations.com
   (no path after the domain for this step)

3) Verify ownership (pick ONE):

   A) DNS TXT record (good if Netlify DNS or registrar manages DNS)
      - Search Console shows a TXT name/value — add it where your DNS is hosted,
        wait a few minutes, click Verify.

   B) HTML file (already in this repo for googleec3213cb4652b237.html)
      - File: public/googleec3213cb4652b237.html → copied to dist/ root on build.
      - After Netlify production deploy (publish directory MUST be dist), open in a
        private window: https://sweitzerautomations.com/googleec3213cb4652b237.html
        You should see one line: google-site-verification: googleec3213cb4652b237.html
      - If that URL is 404, Netlify is not publishing dist from this repo's build;
        fix Build settings (command: npm run build, publish: dist), then redeploy.
      - Then click Verify in Search Console.

   C) HTML tag
      - Google gives <meta name="google-site-verification" content="..."/>
      - Add that meta tag inside <head> in index.html once, deploy, then Verify.

4) After verified: Sitemaps → Add new sitemap → enter: sitemap.xml → Submit.

5) Optional: add the same Google account under Settings → Users if needed.

I cannot run steps 1–4 for you — they require your Google account.
The AI assistant can add the verification file or meta tag after you paste
the exact string Google gives you.
