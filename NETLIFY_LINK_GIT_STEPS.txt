      sweitzer-automations-website
5. Branch to deploy: main
6. Confirm build settings (should match netlify.toml). Do NOT set publish to repo root — must be dist.

IF “Link repository” IS MISSING OR GRAYED OUT

TROUBLESHOOTING BUILD FAILURES
- Build log: ensure Node runs install + build; error “tsc not found” → Netlify runs
  npm install (package.json must list typescript + vite — it does).
- Wrong page on live URL: clear cache or trigger “Clear cache and deploy” once.
